# 모델 가이드 (크레딧 단가는 변동 가능 — 항상 get_cost로 실측)

> 비개발자 사용자는 "더 비싼 모델 = 더 좋은 광고"로 오해하기 쉽다. 실제로는 **포맷에 맞는 모델**이 더 좋은 결과를 낸다. 아래 기준으로 네가 골라서 추천하라.

## 영상 모델 (Plus 플랜 기준, 8초 실측 예시)

| 모델 | id | 8초 비용(예시) | 최대 길이 | 강점 | 언제 |
|------|----|:--:|:--:|------|------|
| **Seedance 2.0** | `seedance_2_0` | ~72(1080p) | 15초 | 레퍼런스 기반 **제품/인물 일관성**, 4K | **기본값.** 제품 광고·UGC, 30초 체이닝의 베이스 |
| Veo 3.1 | `veo3_1` | 22(fast)~ | **8초** | 초사실적 시네마틱 | 히어로·뷰티 클로즈업 (단 8초 한계) |
| Cinema Studio 3.0 | `cinematic_studio_3_0` | ~40 | 15초 | 힉스필드 최고 시네마급 | 브랜드 무드컷·TV 스팟 |
| Kling 3.0 | `kling3_0` | 변동 | 15초 | 멀티샷·오디오·모션전사 | 시네마틱 대안 |

**선택 규칙**
- 제품이 컷마다 똑같이 나와야 한다 → **Seedance 2.0** (레퍼런스 일관성).
- UGC(리얼 후기) → Seedance 2.0. Veo는 너무 영화같아 UGC 환상을 깨므로 피한다.
- 시네마틱 히어로/뷰티 단컷 → Veo 3.1 또는 Cinema Studio 3.0.
- **30초처럼 긴 영상** → Seedance 2.0로 체이닝(8초 한계인 Veo 비권장). `master-prompt.md` 참조.
- 가성비 모델(Kling 3.0 Turbo 등)은 사용자가 "고퀄 원함"을 명시했으므로 기본값에서 제외.

> Veo 3.1은 `quality`(basic/high/ultra)·`model`(preview=최고/fast) 파라미터로 품질을 올리면 비용이 22보다 크게 오른다. 올릴 때는 반드시 get_cost로 재확인.

## 이미지 모델

| 모델 | id | 언제 |
|------|----|------|
| GPT Image 2 | `gpt_image` | **라벨·글자 정확도**가 중요할 때(제품 패키지 텍스트) |
| Nano Banana Pro | `nano_banana_pro` | 최고 품질 사실적 컷 |
| Seedream 5.0 Lite | `seedream_v5_lite` | 레퍼런스 기반 편집, 가벼운 작업 |
| Nano Banana 2 | `nano_banana_2` | 빠른 고품질, 4K |

- 제품 광고컷은 반드시 업로드한 제품 사진을 레퍼런스(role: image)로 넣어 제품 정확도를 유지한다.
- AI는 라벨 텍스트를 변형할 수 있으니, 결과에서 브랜드명·라벨이 원본과 같은지 확인하고 틀어지면 재생성하거나 GPT Image 2로 전환.
