시작하기

비개발자·비영상인도 이 페이지 하나로 고퀄 광고 영상 소재를 만들 수 있게 돕는 보조 도구입니다.

준비: 힉스필드 MCP 연결 (1번만)

1
클로드 → 설정 → 커넥터 → 사용자 지정 커넥터 추가
2
URL 입력 https://mcp.higgsfield.ai/mcp (끝에 /mcp 빠지면 연결 실패)
3
연결 → 힉스필드 로그인(OAuth) 후 권한 "항상 허용".
돈 관련 핵심: 영상 생성은 유료 플랜 필수(무료는 이미지만). 생성 전 항상 크레딧을 확인하세요. Veo·Sora 같은 프리미엄은 크레딧이 빨리 닳습니다.

전체 흐름 5단계

①
컨셉 제품·레퍼런스 분석 → 톤·컬러·메시지·타깃
②
베이스 이미지 제품 사진 레퍼런스로 광고컷 생성
③
프롬프트 빌더·번역기로 정밀 프롬프트
④
영상 생성 시작프레임 물려 끊김 없이 (15초↑ 체이닝)
⑤
발행 자막·카피 얹어 릴스/숏폼

프롬프트 빌더 광고영상 마스터 프롬프트

칸을 채우면 클로드에 붙여넣을 마스터 프롬프트가 자동 완성됩니다.

"더 비싼 모델 = 더 좋은 광고"가 아닙니다. 포맷에 맞는 모델이 정답. 실제 크레딧은 생성 전 확인.

레퍼런스 영상 → 재현 프롬프트

    
"이 영상처럼" 만들고 싶을 때. 영상 분석 → 재현 프롬프트 + 내 제품으로 교체(스왑).

    
진행 순서

      
1
유튜브 링크 또는 영상 파일 (짧을수록 정확, 30초 이내)

      
2
"이 영상처럼 만들어줘" + 링크 → 장면별 분석(3~5분)

      
3
장면표 + 자막/음성 + 목소리 추정 + 재현 프롬프트

      
4
내 제품으로 스왑 → 아래 첨부 가이드대로 업로드

    
제품 스왑 — 형태(폼팩터)가 맞아야 함

      
원본 형태첨부할 제품첨부 사진

        
립스틱/스틱스틱·립·멀티스틱닫힘 정면 / 뚜껑 열어 색 / 45°

        
단지형 크림단지형 크림정면 라벨 / 뚜껑 열어 제형 / 45°

        
튜브형튜브형정면 라벨 / 캡·노즐 / 짜낸 제형

        
펌프/보틀펌프/보틀정면 / 펌프 헤드 / 측면

      
주의: 형태가 다르면(단지→립스틱) 손동작이 깨져 스왑 안 됨. 경쟁사 광고 1:1 복제는 저작권 위험 — 스타일 참고로만.
